用户指导¶

添加按钮¶

首先定义一个回调函数，该回调函数如下所示：

 1 void btn_event(sgl_obj_t *obj, void *data)
 2 {
 3     if(sgl_obj_get_event_status(obj) == SGL_EVENT_PRESSED) {
 4         ...
 5         //your code
 6     }
 7     else if(sgl_obj_get_event_status(obj) == SGL_EVENT_RELEASED) {
 8         ...
 9         //your code
10     };
11 }

然后在main.c函数中添加如下代码用户创建按钮： 1. 静态创建

 1 //定义button样式
 2 sgl_style_t btn_style = {.body_color = SGL_BLUE, .text_color = SGL_RED, .radius = 25};
 3 //定义一个button结构体
 4 sgl_button_t button;
 5 //定义一个button的obj指针
 6 sgl_obj_t *btn = &button.obj;
 7 //按钮初始化
 8 sgl_button_init(&button, sgl_scr_active());
 9 //设置按钮大小，注意：当绘图缓存的大小不大时，按钮的大小不应该设置过大
10 sgl_obj_set_size(btn, 100, 50);
11 //设置按钮的位置，这里使用相对位置
12 sgl_obj_set_align(btn, SGL_ALIGN_CENTER_RIGHT);
13 //设置按钮的样式
14 sgl_obj_set_style(btn, btn_style);
15 //设置按钮的事件回调
16 sgl_obj_set_event_cb(btn, btn_event, NULL);
17 //设置按钮的toggle颜色
18 sgl_button_set_toggle_color(btn, SGL_GREEN);
19 //设置按钮的字体
20 sgl_button_set_font(btn, &song23);
21 //设置按钮的text
22 sgl_button_set_text(btn, "Button");
